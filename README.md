# QuizWit

QuizWit is a Multiple Choice Questions based Quiz application in PHP.

It is designed to be lightweight, easy to use and extendable.

## User Category

1. Admin
2. Candidate

## Admin Panel Features

1. Creating Quiz
2. Update Quiz
3. Expand Quiz
4. Delete Quiz
- Quiz Status
-- Active
-- Inactive
6. Public Access
- Exam Key (conceal, Reveal)
- Evaluation  (conceal, Reveal)
